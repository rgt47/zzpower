# zzpower 0.1.0

* Initial CRAN release
* Interactive Shiny application for power analysis of two-group parallel designs
* Support for multiple effect size specifications (Cohen's d, percentage reduction, difference in change scores, treatment group change)
* Accounts for dropout rates, drop-in rates, and unequal group allocation
* Interactive power curves and detailed results tables
* Downloadable reports in PDF, HTML, and Word formats
* Comprehensive documentation and examples
