# Render a multi-aim study as CSV

Render a multi-aim study as CSV

## Usage

``` r
multi_aim_csv(study, file = NULL)
```

## Arguments

- study:

  A \`multi_aim_study\`.

- file:

  Optional file path. If \`NULL\` returns the CSV content as a character
  string.

## Value

The file path if \`file\` is non-null; otherwise the CSV content as a
string.
