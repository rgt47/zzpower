# Get Effect Size Range for a Test

Standalone utility for generating effect size sequences. Used in tests
and non-module contexts with manually prefixed input IDs.

## Usage

``` r
get_effect_size_range(test_id, input)
```

## Arguments

- test_id:

  Test identifier

- input:

  List or Shiny input with prefixed IDs

## Value

List with effect_sizes, standardized, and method
