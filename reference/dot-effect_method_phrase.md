# Humanise an effect-size method id for narrative use

Humanise an effect-size method id for narrative use

## Usage

``` r
.effect_method_phrase(method, test_id = NULL)
```

## Arguments

- method:

  One of the \`effect_size_methods\` ids declared in the test registry
  (\`"cohens_d"\`, \`"hazard_ratio"\`, etc.).

## Value

A short noun phrase suitable for a methods paragraph.
